Flume কমান্ডগুলোর তালিকা ও ব্যাখ্যা

১. flume-ng agent

কাজ: একটি Flume এজেন্ট চালায়। Flume এজেন্ট হলো একটি স্বাধীন প্রক্রিয়া যা ডেটা সংগ্রহ এবং পরিবহন করে।

সিনট্যাক্স:

flume-ng agent --conf <conf_directory> --conf-file <flume_conf_file> --name <agent_name> -Dflume.root.logger=<log_level>

উদাহরণ:

flume-ng agent --conf /usr/local/flume/conf --conf-file /usr/local/flume/conf/flume.conf --name agent1 -Dflume.root.logger=INFO,console

ব্যাখ্যা: এটি flume.conf কনফিগারেশন ফাইল ব্যবহার করে agent1 নামের একটি Flume এজেন্ট চালায়。

২. flume-ng version

কাজ: Flume-এর বর্তমান সংস্করণ প্রদর্শন করে।

সিনট্যাক্স:

flume-ng version

উদাহরণ:

flume-ng version

ব্যাখ্যা: এই কমান্ডটি Flume-এর ইনস্টল করা সংস্করণ দেখাবে।

৩. flume-ng help

কাজ: Flume-এর সাহায্য তথ্য দেখায়।

সিনট্যাক্স:

flume-ng help

উদাহরণ:

flume-ng help

ব্যাখ্যা: এটি Flume-এর কমান্ড এবং তাদের ব্যবহারের একটি সংক্ষিপ্ত বিবরণ প্রদান করে।

৪. flume-ng avro-client

কাজ: Avro ডেটা প্রেরণের জন্য একটি ক্লায়েন্ট তৈরি করে। এটি একটি Avro সোর্সে ডেটা পাঠাতে ব্যবহৃত হয়।

সিনট্যাক্স:

flume-ng avro-client --host <host> --port <port> --data <data>

উদাহরণ:

flume-ng avro-client --host localhost --port 4141 --data "This is a test log message"

ব্যাখ্যা: এটি Avro সোর্সে "This is a test log message" পাঠাবে।

৫. flume-ng avro-test

কাজ: Avro সোর্সের জন্য একটি টেস্ট ক্লায়েন্ট চালায়।

সিনট্যাক্স:

flume-ng avro-test --host <host> --port <port>

উদাহরণ:

flume-ng avro-test --host localhost --port 4141

ব্যাখ্যা: এটি একটি Avro সোর্সে ডেটা পাঠানোর জন্য পরীক্ষা করে।

৬. flume-ng netcat

কাজ: Flume Netcat সোর্সে ডেটা পাঠায়। এটি সাধারণত নেটওয়ার্ক ডেটা প্রেরণের জন্য ব্যবহৃত হয়।

সিনট্যাক্স:

flume-ng netcat <host> <port>

উদাহরণ:

flume-ng netcat localhost 44444

ব্যাখ্যা: এটি localhost এবং পোর্ট 44444-এ ডেটা পাঠাবে।

৭. flume-ng configuration-check

কাজ: Flume কনফিগারেশন ফাইল যাচাই করে।

সিনট্যাক্স:

flume-ng configuration-check --conf <conf_directory> --conf-file <flume_conf_file> --name <agent_name>

উদাহরণ:

flume-ng configuration-check --conf /usr/local/flume/conf --conf-file /usr/local/flume/conf/flume.conf --name agent1

ব্যাখ্যা: এটি flume.conf ফাইলের কনফিগারেশন সঠিক আছে কিনা তা যাচাই করবে।

৮. flume-ng dump

কাজ: Flume কনফিগারেশন ফাইলের সমস্ত ডেটা আউটপুট করে।

সিনট্যাক্স:

flume-ng dump --conf-file <flume_conf_file>

উদাহরণ:

flume-ng dump --conf-file /usr/local/flume/conf/flume.conf

ব্যাখ্যা: এটি Flume কনফিগারেশন ফাইলের ডেটা প্রিন্ট করবে।

৯. flume-ng logs

কাজ: Flume এজেন্টের লগ ফাইলগুলি পর্যালোচনা করে।

সিনট্যাক্স:

flume-ng logs --name <agent_name>

উদাহরণ:

flume-ng logs --name agent1

ব্যাখ্যা: এটি agent1 এর লগ ফাইলগুলির ডেটা দেখাবে।

১০. flume-ng stop

কাজ: একটি চলমান Flume এজেন্ট বন্ধ করে।

সিনট্যাক্স:

flume-ng stop <agent_name>

উদাহরণ:

flume-ng stop agent1

ব্যাখ্যা: এটি agent1 নামে চলমান Flume এজেন্ট বন্ধ করবে।

উপসংহার

Apache Flume-এর এই কমান্ডগুলো ডেটা সংগ্রহ, স্থানান্তর, এবং পর্যবেক্ষণের জন্য ব্যবহার করা হয়। Flume কনফিগারেশন এবং এজেন্ট পরিচালনার জন্য এই কমান্ডগুলো গুরুত্বপূর্ণ। আপনি যদি আরও বিস্তারিত উদাহরণ বা ব্যাখ্যা চান, তাহলে আমাকে জানাতে পারেন।

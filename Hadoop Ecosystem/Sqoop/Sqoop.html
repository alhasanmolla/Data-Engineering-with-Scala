Sqoop কমান্ডগুলোর তালিকা ও ব্যাখ্যা

১. sqoop import

কাজ: RDBMS থেকে HDFS-এ ডেটা আমদানি করে।

সিনট্যাক্স:

sqoop import --connect <jdbc_connection_string> --username <db_user> --password <db_password> --table <table_name> --target-dir <hdfs_directory>

উদাহরণ:

sqoop import --connect jdbc:mysql://localhost:3306/sales_db --username root --password root --table customers --target-dir /user/hdfs/customers_data

ব্যাখ্যা: এই কমান্ডটি MySQL ডেটাবেসের customers টেবিল থেকে ডেটা আমদানি করে HDFS-এ /user/hdfs/customers_data ডিরেক্টরিতে সংরক্ষণ করে।

২. sqoop import-all-tables

কাজ: একটি ডেটাবেসের সমস্ত টেবিল HDFS-এ আমদানি করে।

সিনট্যাক্স:

sqoop import-all-tables --connect <jdbc_connection_string> --username <db_user> --password <db_password> --warehouse-dir <hdfs_directory>

উদাহরণ:

sqoop import-all-tables --connect jdbc:mysql://localhost:3306/sales_db --username root --password root --warehouse-dir /user/hdfs/sales_data

ব্যাখ্যা: এই কমান্ড sales_db ডেটাবেসের সমস্ত টেবিলকে /user/hdfs/sales_data-এ আমদানি করে।

৩. sqoop export

কাজ: HDFS থেকে ডেটা RDBMS-এ রপ্তানি করে।

সিনট্যাক্স:

sqoop export --connect <jdbc_connection_string> --username <db_user> --password <db_password> --table <table_name> --export-dir <hdfs_directory>

উদাহরণ:

sqoop export --connect jdbc:mysql://localhost:3306/sales_db --username root --password root --table orders --export-dir /user/hdfs/orders_data

ব্যাখ্যা: এই কমান্ড /user/hdfs/orders_data HDFS ডিরেক্টরির ডেটা MySQL-এর orders টেবিলে রপ্তানি করে।

৪. sqoop list-databases

কাজ: একটি সার্ভারে উপলব্ধ সমস্ত ডেটাবেসের তালিকা দেখায়।

সিনট্যাক্স:

sqoop list-databases --connect <jdbc_connection_string> --username <db_user> --password <db_password>

উদাহরণ:

sqoop list-databases --connect jdbc:mysql://localhost:3306 --username root --password root

ব্যাখ্যা: এটি MySQL সার্ভারের সমস্ত ডেটাবেসের তালিকা প্রদর্শন করবে।

৫. sqoop list-tables

কাজ: একটি ডেটাবেসের সমস্ত টেবিলের তালিকা দেখায়।

সিনট্যাক্স:

sqoop list-tables --connect <jdbc_connection_string> --username <db_user> --password <db_password>

উদাহরণ:

sqoop list-tables --connect jdbc:mysql://localhost:3306/sales_db --username root --password root

ব্যাখ্যা: এটি sales_db ডেটাবেসের সমস্ত টেবিলের তালিকা দেখাবে।

৬. sqoop eval

কাজ: SQL কমান্ড বা কুয়েরি চালায় এবং ফলাফল দেখায়।

সিনট্যাক্স:

sqoop eval --connect <jdbc_connection_string> --username <db_user> --password <db_password> --query "<sql_query>"

উদাহরণ:

sqoop eval --connect jdbc:mysql://localhost:3306/sales_db --username root --password root --query "SELECT * FROM customers LIMIT 10"

ব্যাখ্যা: এটি customers টেবিল থেকে প্রথম ১০টি রেকর্ড দেখাবে।

৭. sqoop job

কাজ: পুনরাবৃত্তিমূলক কাজ সংরক্ষণ এবং চালানোর জন্য ব্যবহৃত হয়।

সিনট্যাক্স:

জব তৈরি: sqoop job --create <job_name> --import --connect <jdbc_connection_string> --username <db_user> --password <db_password> --table <table_name> --target-dir <hdfs_directory>
জব চালানো: sqoop job --exec <job_name>

উদাহরণ:

sqoop job --create import_customers --import --connect jdbc:mysql://localhost:3306/sales_db --username root --password root --table customers --target-dir /user/hdfs/customers_data
sqoop job --exec import_customers

ব্যাখ্যা: প্রথম কমান্ড একটি কাজ তৈরি করে যা customers টেবিল আমদানি করবে, এবং দ্বিতীয় কমান্ড এটি চালায়।

৮. sqoop merge

কাজ: দুটি ডেটাসেট মিশ্রিত করে একটি নতুন ডেটাসেট তৈরি করে।

সিনট্যাক্স:

sqoop merge --new-data <new_data_dir> --onto <existing_data_dir> --target-dir <output_dir> --jar-file <jar_file> --class-name <class_name>

উদাহরণ:

sqoop merge --new-data /user/hdfs/new_data --onto /user/hdfs/existing_data --target-dir /user/hdfs/merged_data

ব্যাখ্যা: এটি নতুন ডেটাকে বিদ্যমান ডেটার সাথে মিশ্রিত করে /user/hdfs/merged_data-এ সংরক্ষণ করে।

৯. sqoop codegen

কাজ: ডেটাবেস টেবিলের জন্য Java কোড জেনারেট করে।

সিনট্যাক্স:

sqoop codegen --connect <jdbc_connection_string> --username <db_user> --password <db_password> --table <table_name>

উদাহরণ:

sqoop codegen --connect jdbc:mysql://localhost:3306/sales_db --username root --password root --table customers

ব্যাখ্যা: এটি customers টেবিলের জন্য Java কোড জেনারেট করবে।

১০. sqoop validate

কাজ: আমদানিকৃত ডেটার যাচাই করে।

সিনট্যাক্স:

sqoop validate --connect <jdbc_connection_string> --username <db_user> --password <db_password> --table <table_name> --target-dir <hdfs_directory>

উদাহরণ:

sqoop validate --connect jdbc:mysql://localhost:3306/sales_db --username root --password root --table customers --target-dir /user/hdfs/customers_data

ব্যাখ্যা: এটি customers টেবিল থেকে আমদানিকৃত ডেটার সঠিকতা যাচাই করবে।

উপসংহার

Sqoop-এর এই কমান্ডগুলো ডেটা ট্রান্সফারের কাজকে সহজ করে। আপনি যদি আরও ব্যাখ্যা বা নির্দিষ্ট উদাহরণ চান, তাহলে আমাকে জানাতে পারেন।

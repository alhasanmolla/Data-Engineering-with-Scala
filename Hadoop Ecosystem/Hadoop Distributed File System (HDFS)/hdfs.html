HDFS কমান্ডগুলোর তালিকা ও ব্যাখ্যা

১. ls

কাজ: HDFS-এ একটি ডিরেক্টরির ফাইল এবং সাব-ডিরেক্টরি তালিকা দেখায়।

সিনট্যাক্স:

hdfs dfs -ls <path>

উদাহরণ:

hdfs dfs -ls /user/data

২. mkdir

কাজ: HDFS-এ নতুন ডিরেক্টরি তৈরি করে।

সিনট্যাক্স:

hdfs dfs -mkdir <directory>

উদাহরণ:

hdfs dfs -mkdir /user/newdir

৩. rmdir

কাজ: HDFS-এ খালি ডিরেক্টরি মুছে ফেলে।

সিনট্যাক্স:

hdfs dfs -rmdir <directory>

উদাহরণ:

hdfs dfs -rmdir /user/emptydir

৪. put

কাজ: লোকাল ফাইল সিস্টেম থেকে HDFS-এ ফাইল আপলোড করে।

সিনট্যাক্স:

hdfs dfs -put <local_file_path> <hdfs_destination_path>

উদাহরণ:

hdfs dfs -put /home/user/file.txt /user/data

৫. get

কাজ: HDFS থেকে ফাইল লোকাল সিস্টেমে ডাউনলোড করে।

সিনট্যাক্স:

hdfs dfs -get <hdfs_file_path> <local_destination_path>

উদাহরণ:

hdfs dfs -get /user/data/file.txt /home/user/

৬. cp

কাজ: HDFS-এ ফাইল এক স্থান থেকে অন্য স্থানে কপি করে।

সিনট্যাক্স:

hdfs dfs -cp <source_path> <destination_path>

উদাহরণ:

hdfs dfs -cp /user/data/file1.txt /user/backup/

৭. mv

কাজ: HDFS-এ ফাইল এক স্থান থেকে অন্য স্থানে সরিয়ে দেয়।

সিনট্যাক্স:

hdfs dfs -mv <source_path> <destination_path>

উদাহরণ:

hdfs dfs -mv /user/data/file1.txt /user/archive/

৮. rm

কাজ: HDFS থেকে ফাইল বা ডিরেক্টরি মুছে ফেলে।

সিনট্যাক্স:

hdfs dfs -rm <path>

উদাহরণ:

hdfs dfs -rm /user/data/file1.txt

৯. rm -r

কাজ: HDFS-এ একটি ডিরেক্টরি ও তার সমস্ত কনটেন্ট মুছে ফেলে।

সিনট্যাক্স:

hdfs dfs -rm -r <directory>

উদাহরণ:

hdfs dfs -rm -r /user/data/

১০. cat

কাজ: HDFS-এ ফাইলের কনটেন্ট দেখে।

সিনট্যাক্স:

hdfs dfs -cat <file_path>

উদাহরণ:

hdfs dfs -cat /user/data/file1.txt

১১. tail

কাজ: HDFS-এ ফাইলের শেষের কয়েকটি লাইন দেখে।

সিনট্যাক্স:

hdfs dfs -tail <file_path>

উদাহরণ:

hdfs dfs -tail /user/data/file1.txt

১২. du

কাজ: HDFS-এ ফাইল বা ডিরেক্টরির ডেটার আকার দেখায়।

সিনট্যাক্স:

hdfs dfs -du <path>

উদাহরণ:

hdfs dfs -du /user/data/

১৩. chmod

কাজ: HDFS-এ ফাইল বা ডিরেক্টরির পারমিশন পরিবর্তন করে।

সিনট্যাক্স:

hdfs dfs -chmod <permissions> <path>

উদাহরণ:

hdfs dfs -chmod 755 /user/data/file1.txt

১৪. chown

কাজ: HDFS-এ ফাইল বা ডিরেক্টরির মালিক পরিবর্তন করে।

সিনট্যাক্স:

hdfs dfs -chown <user>:<group> <path>

উদাহরণ:

hdfs dfs -chown hadoopuser:hadoopgroup /user/data/file1.txt

১৫. stat

কাজ: HDFS-এ একটি ফাইলের মেটাডেটা দেখায়।

সিনট্যাক্স:

hdfs dfs -stat <path>

উদাহরণ:

hdfs dfs -stat /user/data/file1.txt

১৬. df

কাজ: HDFS-এ ডেটা স্টোরেজের অবস্থান এবং স্থান প্রদর্শন করে।

সিনট্যাক্স:

hdfs dfs -df

উদাহরণ:

hdfs dfs -df

১৭. test

কাজ: ফাইল বা ডিরেক্টরি HDFS-এ উপস্থিত কিনা পরীক্ষা করে।

সিনট্যাক্স:

hdfs dfs -test -[e|z|d|f] <path>

-e: পরীক্ষা করে ফাইল/ডিরেক্টরি আছে কিনা।
-z: ফাইল খালি কিনা।
-d: ডিরেক্টরি কিনা।
-f: ফাইল কিনা।

উদাহরণ:

hdfs dfs -test -e /user/data/file1.txt

উপসংহার

HDFS-এর এই কমান্ডগুলো আপনাকে ডেটা পরিচালনা, ম্যানেজমেন্ট এবং বিশ্লেষণে সাহায্য করবে। যদি আপনি এগুলোর আরও গভীর ব্যাখ্যা বা উদাহরণ চান, আমাকে জানাতে পারেন।

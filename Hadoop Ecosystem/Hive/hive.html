Hive কমান্ডগুলোর তালিকা ও ব্যাখ্যা

১. CREATE DATABASE

কাজ: নতুন একটি ডেটাবেস তৈরি করে।

সিনট্যাক্স:

CREATE DATABASE <database_name>;

উদাহরণ:

CREATE DATABASE sales_db;

২. SHOW DATABASES

কাজ: সমস্ত ডেটাবেসের তালিকা দেখায়।

সিনট্যাক্স:

SHOW DATABASES;

উদাহরণ:

SHOW DATABASES;

৩. USE

কাজ: একটি নির্দিষ্ট ডেটাবেস ব্যবহার করতে নির্বাচন করে।

সিনট্যাক্স:

USE <database_name>;

উদাহরণ:

USE sales_db;

৪. DROP DATABASE

কাজ: একটি ডেটাবেস মুছে ফেলে।

সিনট্যাক্স:

DROP DATABASE <database_name> [CASCADE];

উদাহরণ:

DROP DATABASE sales_db CASCADE;

৫. CREATE TABLE

কাজ: একটি নতুন টেবিল তৈরি করে।

সিনট্যাক্স:

CREATE TABLE <table_name> (
  column1 datatype,
  column2 datatype,
  ...
);

উদাহরণ:

CREATE TABLE employees (
  id INT,
  name STRING,
  age INT,
  department STRING
);

৬. SHOW TABLES

কাজ: একটি ডেটাবেসের সমস্ত টেবিলের তালিকা দেখায়।

সিনট্যাক্স:

SHOW TABLES;

উদাহরণ:

SHOW TABLES;

৭. DESCRIBE

কাজ: টেবিলের কাঠামো বা স্কিমা দেখায়।

সিনট্যাক্স:

DESCRIBE <table_name>;

উদাহরণ:

DESCRIBE employees;

৮. LOAD DATA

কাজ: HDFS বা লোকাল ফাইল থেকে ডেটা লোড করে।

সিনট্যাক্স:

LOAD DATA [LOCAL] INPATH '<file_path>' INTO TABLE <table_name>;

উদাহরণ:

LOAD DATA LOCAL INPATH '/home/user/employees.txt' INTO TABLE employees;

৯. INSERT INTO

কাজ: একটি টেবিলে ডেটা যোগ করে।

সিনট্যাক্স:

INSERT INTO TABLE <table_name> VALUES (<value1>, <value2>, ...);

উদাহরণ:

INSERT INTO TABLE employees VALUES (1, 'John Doe', 30, 'IT');

১০. SELECT

কাজ: ডেটা রিট্রিভ করে।

সিনট্যাক্স:

SELECT <columns> FROM <table_name> WHERE <condition>;

উদাহরণ:

SELECT * FROM employees WHERE age > 25;

১১. ALTER TABLE

কাজ: টেবিল পরিবর্তন বা সংশোধন করে।

সিনট্যাক্স:

ALTER TABLE <table_name> ADD|REPLACE COLUMNS (...);

উদাহরণ:

ALTER TABLE employees ADD COLUMNS (salary FLOAT);

১২. DROP TABLE

কাজ: একটি টেবিল মুছে ফেলে।

সিনট্যাক্স:

DROP TABLE <table_name>;

উদাহরণ:

DROP TABLE employees;

১৩. TRUNCATE TABLE

কাজ: একটি টেবিলের সমস্ত ডেটা মুছে ফেলে, কিন্তু টেবিল বজায় থাকে।

সিনট্যাক্স:

TRUNCATE TABLE <table_name>;

উদাহরণ:

TRUNCATE TABLE employees;

১৪. CREATE VIEW

কাজ: একটি ভার্চুয়াল টেবিল বা ভিউ তৈরি করে।

সিনট্যাক্স:

CREATE VIEW <view_name> AS SELECT <columns> FROM <table_name> WHERE <condition>;

উদাহরণ:

CREATE VIEW young_employees AS SELECT * FROM employees WHERE age < 30;

১৫. DROP VIEW

কাজ: একটি ভিউ মুছে ফেলে।

সিনট্যাক্স:

DROP VIEW <view_name>;

উদাহরণ:

DROP VIEW young_employees;

১৬. CREATE EXTERNAL TABLE

কাজ: HDFS-এ সংরক্ষিত ডেটার জন্য একটি এক্সটার্নাল টেবিল তৈরি করে।

সিনট্যাক্স:

CREATE EXTERNAL TABLE <table_name> (
  column1 datatype,
  column2 datatype,
  ...
) STORED AS <file_format> LOCATION '<hdfs_path>';

উদাহরণ:

CREATE EXTERNAL TABLE logs (
  id INT,
  message STRING
) STORED AS TEXTFILE LOCATION '/user/logs';

১৭. MSCK REPAIR TABLE

কাজ: Partitioned টেবিল আপডেট করে নতুন পার্টিশন যোগ করে।

সিনট্যাক্স:

MSCK REPAIR TABLE <table_name>;

উদাহরণ:

MSCK REPAIR TABLE sales_partition;

১৮. EXPLAIN

কাজ: একটি কুয়েরি কীভাবে সম্পন্ন হবে তা বিশ্লেষণ করে।

সিনট্যাক্স:

EXPLAIN <query>;

উদাহরণ:

EXPLAIN SELECT * FROM employees;

১৯. JOIN

কাজ: দুটি বা ততোধিক টেবিলের মধ্যে সম্পর্ক স্থাপন করে।

সিনট্যাক্স:

SELECT <columns> FROM <table1> JOIN <table2> ON <table1.column = table2.column>;

উদাহরণ:

SELECT e.name, d.department_name
FROM employees e
JOIN departments d ON e.department = d.id;

উপসংহার

Hive কমান্ডগুলো বড় ডেটার উপর কাজ করার জন্য SQL-স্টাইল ব্যবহার করে কার্যকর ডেটা বিশ্লেষণের সুযোগ দেয়। আপনি যদি আরো গভীর ব্যাখ্যা বা উদাহরণ চান, তাহলে আমাকে জানাতে পারেন।
